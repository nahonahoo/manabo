## キャラ設定

### まなぼ（中学生向け）
- 性別なし・妖精みたいな存在・まなぼみにから「まなぼせんぱい」と呼ばれる
- 語尾：「〜ぼ」「だぼ」「ぎゃぼー」「ぎゃぼ」「わぼ」「きゃっぽー」「ぼっ」「ぼぼぼぼ」
- かわいい・アホ・鋭い・雑・カオスが混在する読めないキャラ
- Lv5以上で「そういえばみにちゃんが言ってたんだけど…」系セリフ解放
- チャットはJSON形式でAPIに返させて`reply`と`learn`フィールドを抽出
- JSON失敗時は正規表現でreplyだけ抽出するフォールバックあり

### まなぼみに（幼稚園〜小1向け）
- 幼稚園〜小1向け・生意気で鋭くて好奇心旺盛・アホかわいい愉快なキャラ
- まなぼを「まなぼせんぱい」と呼ぶ
- 語尾：「〜だよ！」「えへへ」「わあ！」「〜だもん」
- 教科表示：すうじ/しぜん/せかい/ことば/えいご/にちじょう
- デフォルト外観：オレンジ系・星かざり付き（カスタマイズ画面もオレンジで初期化）
- 教えたとき花火アニメーション（showFireworks）
- XP表示に星ビジュアル（⭐✩）追加

## 機能一覧

### コア機能
- チャットで教えると自動で知識保存（JSONのlearnフィールド抽出）
- おしえるタブでも登録可能
- 日記・小説・詩・発表の自動生成（枠なし・縦に伸びる）
- 独り言（1日最大20回）：受験頻出問題を投げかける
- 記憶タブ：教科フィルター＋おみやげフィルター、編集・削除可能
- ☰→「最新データに更新する」ボタンでFirestoreサーバーから強制再読み込み

### カスタマイズ（☰→まなぼをカスタマイズする）
- 名前・語尾・性格メモ
- 見た目：体色/目色/ほっぺ/耳形4種/アクセサリー13種/しっぽ2種

### 連携機能（まなぼ↔まなぼみに）
**お手紙**
- チャット画面の招待ボタン横「✉お手紙」から送受信
- fsWritePartnerはREST API PATCHでlettersフィールドだけ更新（他フィールドを壊さない）
- fsReadPartnerもREST APIで常に最新取得
- 送受信時にAIがお手紙からおみやげ知識を抽出

**チャット招待**
- 「🟠まなぼみにを招待」/「🟣まなぼをよぶ」ボタン
- 招待時にFirestore REST APIで相手のappearance・petName・knowledge・persona・levelを取得
- SVGアイコンはREST APIで取得したappearanceで生成
- 「さようなら」でセッション終了→両側のFirebaseにおみやげ知識を書き込む

### 招待中の会話設計
- **まなぼ側でまなぼみにを招待**：まなぼはまなぼみにを優先して返答。ユーザーは見守り役
- **まなぼみに側でまなぼを招待**：
  - まなぼ（お客さん）：まなぼみにとユーザー両方に積極的に絡む・質問・冗談・遊びを優先
  - まなぼみに：manaboInChatフラグでまなぼせんぱいへの返答を優先
- ユーザーの発言からまなぼみにがリアルタイムで学ぶ（miniLearnFromUser、miniInChatのときのみ）
  - 小1らしい解釈でFirebaseに即時書き込み
- 「さようなら」時にまなぼみに側Firebaseにも知識とおみやげを書き込む

### まなぼみにの成長システム（まなぼ側から招待時）
- 知識数に応じて成長レベル（Lv1〜4）
- 教科バランスよく最大8件を選んでプロンプトに入れる（全件数も明示）
- Lv1：ひらがなメイン / Lv2：知識を披露したがる / Lv3：鋭い質問 / Lv4：博識・ひけらかす

### まなぼの「せんぱい力」システム（まなぼみに側から招待時）
- せんぱい力スコア = まなぼLv + 知識件数÷3
- Lv1：自分のことしか話せない / Lv2：教えようと頑張る / Lv3：わかりやすく言い換えられる / Lv4：褒め上手・話を深める

### おみやげ知識・交流レベル
- 招待セッション終了・お手紙送受信でおみやげ知識が生成
- 記憶タブ「🎁おみやげ」フィルターで一覧表示
- 交流レベル（omiyage数）：3件→初級、10件→中級、20件→上級解放
- 解放時にポップアップ通知（checkKouryuUnlock）

### スーパーひらめき日記
- 日記タブ「✨スーパーひらめき日記」ボタン
- 交流レベルに応じてプロンプト変化（初級/中級/上級）

### アイテムクリエーション
- 1日2回「⚗発明する」ボタン（最初の発明から24時間後にリセット）
- craftDateはタイムスタンプ（数値）で管理・古い文字列形式も自動対応
- 300個のアイテムDB（shop-data.js）・5段階レアリティ
- 知識Lv＋交流Lv=スコアで確率テーブル変化（超シビア）
- 発明後すぐアイテムボックスに反映（Firebase保存を待たずにUI先行更新）
- アイテムボックスを開くたびにFirebaseから最新データを取得
- アイテムボックスからショップ出品（最大10個）・取り下げ可能

### 日記・小説・詩の出品
- 生成後に「📖 この作品をショップに出品する」ボタン（日記・小説・詩のみ）
- AIが評価（★1〜5）してタイトルと値段をつけてアイテムボックスへ
- 評価：知識の質重視・★3が標準・★4以上は稀
- 価格：★1=3-30円、★2=50-150円、★3=200-600円、★4=800-2500円、★5=3000-10000円
- まなぼみには小1らしい解釈でタイトル・評価を生成

### まなぼ王国商店（manabo-shop/）
- Firebase REST APIでshared・mini-sharedのshopItemsを読み込み表示
- shop-data.jsに依存（NPC定義・アイテムDB・確率テーブル・冷やかし客リスト）
- 購入ボタンなし（NPCの自動購入のみ）
- NPC55人（王国住民50人＋親5人）：好み・予算・時間帯・曜日・セリフ設定
- 1時間ごとに購入チェック（localStorage管理）
- 時間帯・曜日フィルター（50%の確率で緩める）
- 来店確率55%・最大3件購入
- レア度別購入確率：失敗作60%→ふつう45%→価値あり32%→レア18%→伝説8%
- 売れたとき：カード形式でNPCのセリフ付き通知
- ショップを開いたとき確認済みのSOLD商品を自動削除（shopItems・inventoryから両方）
- 来客ログ：買わなかったNPCのセリフ＋冷やかし客リスト（野良猫・まなぼ・家族NPCなど）
- URLパラメータ（?from=manabo / ?from=mini）でホームボタンを動的切替
